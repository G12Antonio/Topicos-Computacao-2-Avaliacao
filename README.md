Atividade Avaliativa 3 (AV2) - Estudo e Implementação sobre Mecanismos de Atenção e Redes Transformers:
Nesta atividade, foi apresentado o código sobre como funciona os mecanismos de atenção e redes transformers. O código usa 5 baselines (Majority class, TF + IDF LogReg (SGD), TF + IDF Multinomial NB, TF + IDF Linear SVM e BiLSTM), 4 modelos de transformers [RoBERTa - base (modelo padrão), RoBERTa + LoRA (PEFT) (modelo variante), RoBERTa + Ablações (duas configurações de ablações diferentes)]
